# @codecheck/output-report

Report output plugin for CodeCheck. Writes machine-readable JSON and human-readable HTML reports after every run.

## Output

- `.codecheck-results/report.json` — structured results for CI/tooling
- `.codecheck-results/report.html` — styled HTML report for sharing

Part of the [CodeCheck](https://github.com/medhavee-upadhyaya/codecheck) ecosystem.
