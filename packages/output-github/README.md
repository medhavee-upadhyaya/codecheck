# @codecheck/output-github

GitHub output plugin for CodeCheck. Posts test results as PR comments and GitHub Actions annotations.

## Features

- PR comment with pass/fail summary
- GitHub Actions annotations on failing functions
- Step summary output

Used automatically by `@codecheck/trigger-ci` when running in GitHub Actions.

Part of the [CodeCheck](https://github.com/medhavee-upadhyaya/codecheck) ecosystem.
