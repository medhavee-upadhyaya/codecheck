# @codecheck/output-terminal

Terminal output plugin for CodeCheck. Renders colored pass/fail results with plain-English error explanations.

## Features

- Colored output (green pass, red fail)
- Plain-English error messages via `humanizeError()`
- Technical error details shown as dimmed secondary line
- Grouped by file

Used automatically by all trigger plugins.

Part of the [CodeCheck](https://github.com/medhavee-upadhyaya/codecheck) ecosystem.
