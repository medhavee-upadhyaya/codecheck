# @codecheck/trigger-onsave

File watcher trigger for CodeCheck. Watches for file changes and runs AI-generated tests on save.

## Usage

```bash
npx codecheck-watch
npx codecheck-watch --dry-run
```

Watches `.ts`, `.tsx`, `.js`, `.jsx`, and `.py` files. Tests run automatically when a file is saved.

Part of the [CodeCheck](https://github.com/medhavee-upadhyaya/codecheck) ecosystem.
