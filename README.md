# CodeCheck

**Your code tests itself.**

An AI-powered, modular, zero-config self-testing ecosystem for developers. One install. No test files to write. No framework to learn.

```bash
npm install @codecheck/core @codecheck/trigger-oncommit @codecheck/scope-unit @codecheck/scope-smoke @codecheck/output-terminal
```
